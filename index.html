Mes projets Python

Gestion de la base de données d'une université

Cette interface web créé en Python permet de gérer une base de données d'une Université. Cette interface permet de visualiser les données, de voir à quelles matières un élève est inscrit, de générer des bulletins... Les données présentes dans cette base ont été obtenu par parsing de fichier CSV. Ce projet a été réalisé pendant l'été 2021.

Projet PNR

Le projet PNR est mon plus gros projet réalisé à ce jour. Ce projet à destination du Parc Naturel Régional du Golfe du Morbihan (PNR) avait pour but de développer un logiciel de gestion de la base de données du PNR. Ce logiciel a été réalisé en Java FX entre février et juin 2022.
