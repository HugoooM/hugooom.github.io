Mes projets Python

Gestion de la base de données d'une université

Cette interface web créé en Python permet de gérer une base de données d'une Université. Cette interface permet de visualiser les données, de voir à quelles matières un élève est inscrit, de générer des bulletins... Les données présentes dans cette base ont été obtenues par parsing de fichier CSV. Ce projet a été réalisé pendant l'été 2021.

Incrustation d'une image

Ce projet, consiste à créer une IHM permettant d'incruster une image dans une autre. Par exemple, incruster un personnage devant un paysage. Ce projet a été réalisé en avril 2021.

Floutage d'une image

Ce script Python consiste à créer une interface graphique permettant de flouter une image, selon plusieurs niveaux de flou possible. Ce projet a été réalisé en mars 2021.

Mes projets Java

Projet PNR

Le projet PNR est mon plus gros projet réalisé à ce jour. Ce projet à destination du Parc Naturel Régional du Golfe du Morbihan (PNR) avait pour but de développer un logiciel de gestion de la base de données du PNR. Ce logiciel a été réalisé en Java FX entre février et juin 2022. Ce projet est disponible sur GitHub.

Jeu de Marienbad

Le jeu de Marienbad est un jeu du type jeux de Nim. Tour à tour, les joueurs doivent retirer des allumettes, le joueur qui retire la dernière allumettes perd. 2 versions ont été créées, la première joueur contre joueur et la seconde est une version joueur contre machine. Ce projet à été réalisé en novembre 2021. Ce projet est disponible sur GitHub.

Mes autres projets

Balabox

Le projet Balabox est un projet initié par Collège numérique 56, qui a pour but de développer l'usage des équipements numériques à des fins pédagogiques. Ce projet Balabox consiste à créer un système de partage, de création, et de diffusion de contenu numérique en classe. Concrètement une Balabox est un Raspberry offrant de nombreux services à des téléphones via un réseau Wifi. Ce projet est disponible sur GitLab.
